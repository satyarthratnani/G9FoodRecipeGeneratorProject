DESCRIPTION

Our objective is to create an application that will assist an end-user tofind the recipe and ingredients of any food by name or an image. Thesteps to be followed are:

  • To explore various related datasets and perform datapreprocessing in order to extract the desired attributes.
  • To perform model training using Machine Learning or DeepLearning algorithms to find the food name.
  • Using spoonacular and Edaman Recipe search & Diet APIs findthe best matching recipes corresponding to a given food .
  • Find food related information from youtube and other website also show on the feed of the user.

FEATURES

  • We aim to make a user aware of the various dishes which can be cooked fromavailable set of ingredients or an input image given by a user. Thus, systems like this never get their place in real life.
  • We deciding to developeda Recipe Generation model by applying Neural Network on it. The tools used fordeveloping the project are Python, React, CSS, JavaScript & Django. A dynamic website incorporating machine learning techniques. Search options:
    • Search by name
    • Search by food image

PROBLEM & SOLUTION

  • Problem: Sometimes people are in a state of dilemma that how to make a foodwhich are delicious in taste and don't know about recipe of food item.
  • Solution: we have design one stop solution where you have to just put image ofdish you want to make or just enter the name of that recipe and byproceeding further you would get the steps to make that recipe and youcan also search according to categories this are the reasons whyactually need of this system.

FUTURE PLANS
